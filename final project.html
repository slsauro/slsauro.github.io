Final Project

For my final programming project I decided to create something anyone can use their own creativity on. In this app you can can decorate your very own christmas tree once you press the button to start. On the side of the screen you should see several color options to choose from. You can use any of these buttons to change the colors and add any decorations and designs that you'd like to the tree such as ornaments, lights or presents. This app is easy and fun to use and is all age appropriate. Enjoy decorating!
